History of projects


History.

2017
PHP DevelStudio was the first "IDE" where i started to learn programming
That "IDE" used PHP as programming language
The first time when i've heard about PHP DevelStudio it was like the beginning of the 2017
Seeing how easy it is to create a program (which can, however, can be easily decompiled)
I was thinking to create a few programs
I forgot about PHP DevelStudio until November 2017, i was decompiled the first program
On 1th December 2017, i was thinking to create launcher, and i made a few updates and gave it up

The rising.

2018

In the beginning of 2018 like January, i've created Calculator, Task Manager (that can only kill process though), Base64 Encoder/Decoder, and i've created Unlocker
In the summer of 2018, i've created the first decompiler VDecompiler that decompiles basic sections
In the autumn of 2018, i've created another decompiler, but he could decompile Protectors, except for specific
In the middle autumn 2018, i've created VDecompiler 2.0 and i've learned a lot about DevelStudio Protection
In October 2018, i've created "obfuscator" program that was replacing strings to "IIIIIII", or "OOOOOOOO", "eval(base64_decode("
In the end of 2018, i've created "useless" the program that was used to test something (i can't remember)
It was the end of 2018, i was thinking what to create, and it was TrashLauncher 2.0 (i still didn't know how to write a program completely and took a lot from the internet)

The start to falling.

2019

Can't say what happened in the beginning of 2019 and middle 2019
In the summer of 2019, i've created TrashLauncher 2.0, but he had a lot of bugs, and his protection was not a good
In the end of 2019, and as always i've created TrashLauncher 2.0 (I don't know what for i've created him then)

The falling.

2020

Starting from this year i got how do i can decompile any program that was made by PHP DevelStudio
The year when i decided to leave PHP DevelStudio compelety, because community were switching to DevelNext or change to the other programming language
And then i've learned Java, so i can write programs on Java
In the end of 2020, i've created TrashLauncher 2.0 (UPD: There's a new launcher called TrashLauncher 2.0 written using C# + C++)

History.

2018
I tried DevelNext in January 2018, as usual i didn't know how to write a little complex program than hello world (lol)
For sure, DevelNext is definitely better than PHP DevelStudio at compiling the classes to bytecode thereby making almost impossible to decompile the code (There was one decompiler, but he doesn't work since DevelNext added protection)
In spring 2018, i've created Calculator 2.0 with a new interface and he was supported cross-platform running (you could run him on Windows, Linux, MacOS)
In summer 2018 or so, i've learned a lot about DevelNext (History, Syntax, Protection, Compiling, Ui)
The following is not without TrashLauncher that was created on DevelNext with a new dark interface and a good protection from decompilers
TrashLauncher existed until the end of 2018

Status of projects: Calculator 2.0: Discounted: 2018.03 (2018/3), TrashLauncher: Discounted: 2018.12 (2018/12)

It could have been a good IDE for creating cross-platform programming, but i didn't like syntax
